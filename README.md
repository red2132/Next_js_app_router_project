# Next.js 기반 도서 프로젝트
## 목표
이 프로젝트는 각각 page Router 버전, app Router 버전으로 제작하며, 그에 따른 Next의 차이와 기능을 이해해 본다.

## 개발환경
언어: Typescript
프레임워크: Next.js 15버전
백엔드: prisma(지원 받음)
